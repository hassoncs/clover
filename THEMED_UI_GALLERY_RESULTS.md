### Generation Stages

Each control flows through 3 stages:

1. **Base State Generation** (`uiBaseStateStage`)
   - Create silhouette (SVG ‚Üí PNG)
   - Upload to Scenario.com
   - Generate base state via img2img (strength 0.95)
   - Remove background
   - Save to artifacts

2. **Variation States** (`uiVariationStatesStage`)
   - For each non-base state
   - Use base state as reference
   - Generate via img2img (strength 0.7)
   - Remove background
   - Save to artifacts

3. **R2 Upload** (`uiUploadR2Stage`)
   - Upload all state images to R2
   - Create metadata.json with state mappings
   - Return public URLs and R2 keys
